# Customer Satisfaction Prediction

## Overview
This project focuses on predicting airline customer satisfaction using machine learning. The primary objective was not only to build a predictive model but also to apply and practice data preparation techniques that are required in real-world data science projekts.

Thr project covers the complete workflow from data understanding and preprocessing to model development and evaluation.

## Dataset
The dataset contains airline customer information and flight-related attributes, including:
- Airline information
- Destination characteristics
- Departure delays
- Waiting times
- Cleanliness rating
- Safety ratings
- Customer satisfaction labels

### Target Variable
- Satisfied
- Not Satisfied

## Conclusion
The project demonstrated how critical data preparation is for successful machine learning applications.
Although machine learning models were able to identify satisfies customers with high accuracy, the minotity class remained difficult to predict due to class imbalance within the dataset.
The results emphasize that model performance must always be interpreted in the context of the underlying data distribution. Techniques such as SMOTE can help mitigate imbalance issues, but they can not fully compensate for a lack of representative minority-class observations.
Beyond predictive performance, the project strengthened practical skills in data preprocessing, feature engineering and building machine learning workflows.
